# Vincent's Word Studies in the New Testament

This directory contains the text and parsed data for Marvin R. Vincent's "Word Studies in the New Testament".

## Content

- `vincent_word_studies.txt`: The raw text of the 4-volume work, downloaded from the Internet Archive.
- `vincent_word_studies_data.json`: The parsed, structured data, organized by book, chapter, and verse.
- `metadata.json`: Provides metadata about the work, including author, publication date, and source.

## Usage

The data in this module is loaded and used by the `EnhancedLexiconHelper` class, providing etymological and word study insights for New Testament verses.
