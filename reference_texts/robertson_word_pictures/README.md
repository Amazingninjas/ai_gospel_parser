# Robertson's Word Pictures in the New Testament

This directory contains the text and parsed data for A.T. Robertson's "Word Pictures in the New Testament".

## Content

- `robertson_word_pictures.txt`: The raw text of the 6-volume work, downloaded from the Internet Archive.
- `robertson_word_pictures_data.json`: The parsed, structured data, organized by book, chapter, and verse.
- `metadata.json`: Provides metadata about the work, including author, publication date, and source.

## Usage

The data in this module is loaded and used by the `EnhancedLexiconHelper` class, providing commentary and grammatical insights for New Testament verses.
