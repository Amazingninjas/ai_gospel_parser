================================================================================
AI GOSPEL PARSER - Installation Instructions
================================================================================

Thank you for downloading the AI Gospel Parser!

This is a comprehensive Biblical Greek study tool with AI-powered analysis.


CHOOSE YOUR INSTALLER:
----------------------

**Windows Users:**
  Double-click: INSTALL.bat

**macOS Users:**
  Right-click INSTALL.command → "Open With" → Terminal
  (You may need to allow it in System Preferences → Security & Privacy)

**Linux Users:**
  Open terminal in this folder and run:
  bash INSTALL.sh


WHAT HAPPENS DURING INSTALLATION:
----------------------------------

1. System requirements check (Python, RAM, disk space)
2. Recommendation for AI provider (Ollama local or Google Gemini API)
3. Automatic dependency installation
4. Download English Bible translation (WEB)
5. Create desktop shortcuts for easy access


SYSTEM REQUIREMENTS:
--------------------

**Minimum (Gemini API):**
- Python 3.8+
- 4 GB RAM
- 2 GB disk space
- Internet connection

**Recommended (Ollama Local - Private):**
- Python 3.8+
- 16 GB RAM
- 8 GB VRAM (GPU optional)
- 20 GB disk space


FEATURES:
---------

- Greek New Testament (SBLGNT) with morphological tagging
- World English Bible (WEB) interlinear translation
- Strong's Greek Lexicon integration
- Verse-by-verse navigation
- AI-powered text analysis (your choice of provider)
- 100% private option with local AI (Ollama)


GETTING HELP:
-------------

After installation completes:
1. Read GETTING_STARTED.txt for comprehensive guide
2. Read QUICK_START.md for quick reference
3. Check install_log.txt if you encounter issues


FIRST TIME USERS:
-----------------

The installer is completely automated and will guide you through:
- Checking if your system meets requirements
- Helping you choose the right AI provider
- Installing everything you need
- Setting up desktop shortcuts

Just run the installer for your platform and follow the prompts!
